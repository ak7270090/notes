## Tech Stack

Client: <b>`React JS`</b>

Server: <b>`Node.js`</b>, <b>`Express.js`</b>

Database: <b>`Mongo DB`</b>

<br>

## Features
- <b>`Authentication`</b>
  - Log In
  - Sign Up
- <b>`User Profile`</b>
- <b>`Search Notes using title`</b>
- <b>`Note`</b>
  - Create Note
  - Edit Note
  - Delete Note
